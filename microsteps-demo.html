HTML

Hypertext Markup Language.

HTML defines the information or data.
eg : paragraph, heading, image, table, list.

There are three parts in writing HTML.

  • tag: specifies the type of the content.
  • attribute: gives more information about the tag.
  • children: body of the tag.
CSS
Cascading Style Sheet

CSS styles the information.
Eg: Color and size of font, background color, border etc

CSS also specify the layout of the website.

JavaScript

Javascript adds interactivity to your application.

It captures user input and dynamically change the content.

It fetches information from server and display.

Recent times, we write most of the business logic in javascript, and reduce the load on the server.

It means we can save money.

HTML5

Whatever present in HTML topic plus lots of new feature.

Lots of new input types. Eg : Color, email, range

Lots of new semantic tags: Eg: header, article, footer

Support for Video and Audio.

Advanced graphic support through SVG and Canvas.

CSS3

Whatever present in CSS topic plus lots of new feature.

Supports rounded corners, gradient, shadows.

Support for animation effects and time bound animations.

New tools for layouting like Flex box.

ES6

The latest version of JavaScript. Here we will look at new features.

Lot of new keywords borrowed from other programming languages: Eg: class, extends, static, const

Support for block level scope, in addition to the default function level scope.

More options when when working with arrays and collections.

And more advanced level features.
